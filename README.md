# CobaltPDF Issue Tracker

This repository is the public issue tracker for [CobaltPDF](https://cobaltpdf.com) — a fast, modern C# library for converting HTML to PDF in .NET.

Use this repo to:

- 🐛 **Report a bug** — something not working as expected
- 💡 **Request a feature** — suggest an improvement or new capability
- ❓ **Ask a question** — get help with usage or configuration

## Before opening an issue

1. Check the [Troubleshooting guide](https://cobaltpdf.com/Troubleshooting) for common solutions
2. Search [existing issues](https://github.com/CobaltPDF/cobaltpdf-issues/issues) to avoid duplicates
